#Books Scraper

A Python web scraping project that extracts book data from a website.

## Features
- Scrapes multiple pages
- Extracts:
  - Title
  - Price
  - Availability
  - Rating
- Exports data to:
  - CSV
  - JSON

## Technologies
- Python
- Requests
- BeautifulSoup

## Usage

```bash
pip install -r requirements.txt
python main.py
